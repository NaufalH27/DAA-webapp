# Cara Menjalankan Demo di localhost

1. Clone github repo
```bash
git clone https://github.com/NaufalH27/DAA-webapp
```

2. cd code directory
 ```bash
cd DAA-webapp
```  

3. install dependencies
```bash
npm install
```

4. Jalankan Program
```bash
node src/app.js
```


5. Buka browser dan Akses URL  this url:
```bash
http:\\localhost:8000
```
- note : bisa di ubah di app.js
